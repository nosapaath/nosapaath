My Work

Instanews

A responsive API based web application, powered by NY Times API.

The Process

I started with analyzing the data object that we get from the API. Find out which paths we need, to retrive the data that we want; a url for picture, link to the article, and a blurb of the news. Use Javascript to assign the paths to variables, and using a loop to inject the mark up that I want to display the news blocks.

What I learned

  • Working with API
  • CSS Flex boxes
  • Mobile First Approach
  • Javascript
  • Gulp to compile JS and Sass files
